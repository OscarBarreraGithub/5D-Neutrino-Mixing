# Deep Research May 16 External Review Against Flavor Catalog v0.1

I read `flavor_catalog/external_research/deepresearch_may16.txt` end to end, including the references block through line 2091. I compared it with the 75-row fact-check inventory, the plan-v1 128-process seed list, DA-4 convergence, Opus round-1 cross-cutting comments, and representative process files across kaon, charm, beauty, top/Higgs/EW, charged-lepton, and EDM families. The catalog's current fact-check status is strong: 74 `VERIFIED`, 1 `PARTIAL`, 0 value-bearing mismatches (`flavor_catalog/audits/factcheck_status.md:110-128`). The main discrepancies below are therefore mostly scope and freshness issues, not evidence that the existing 75 entries are numerically unreliable.

## Section 1: Processes / Observables in the External Research That We Are Missing

1. **Radiative-helicity B observables: `A_CP(B -> X_s gamma)`, `S_{K*gamma}`, `B_s -> phi gamma` CP/helicity observables.** The external report explicitly separates these from the inclusive rate and says right-handed dipoles are "far more visible here than in the total rate alone" (`deepresearch_may16.txt:664-721`). Our catalog has B011 for inclusive `B -> X_s gamma`, with the process file explicitly saying it is separated from exclusive `B -> K* gamma` and `B_s -> phi gamma` modes (`flavor_catalog/processes/beauty/B011.tex:13-19`). Plan v1 had B012/B013/B014 for exclusive radiative variants (`docs/phase_logs/flavor_catalog_plan_v1.md:273-276`), and DA-4 deferred them as "exclusive radiative variants after inclusive b->s gamma" (`flavor_catalog/worklogs/discovery/round_004_convergence.md:35`). Judgment: the deferral is only partly sound. For a compact 75-process catalog, the inclusive rate is the stronger anchor. But for warped flavor with split LH/RH localization, the external research is right that helicity observables are diagnostic. At least one exclusive radiative/helicity row should be reopened or the deferred-scope rationale should say explicitly that helicity information is being sacrificed.

2. **Top photon FCNC decays `t -> u gamma` and `t -> c gamma`.** The external report lists current PDG-style limits for `t -> q gamma` alongside `t -> qg`, namely `B(t -> u gamma) < 0.85 x 10^-5` and `B(t -> c gamma) < 4.2 x 10^-5` (`deepresearch_may16.txt:1339-1346`). Plan v1 included T003 and T004 (`docs/phase_logs/flavor_catalog_plan_v1.md:307-308`), but the final 75-process inventory does not include them (`flavor_catalog/worklogs/discovery/round_004_convergence.md:13`), and DA-4's deferred Top/Higgs/EW list does not name T003/T004 (`flavor_catalog/worklogs/discovery/round_004_convergence.md:36`). Judgment: this is a catalog bookkeeping mistake. These are secondary compared with `t -> qZ`, `t -> qH`, and `t -> qg`, but they were in the plan and the external research correctly treats them as part of the top-dipole set.

3. **Top-Higgs FCNC `t -> uH`.** The external report groups `t -> uH` with `t -> cH` and quotes a limit scale of `B(t -> uH) < 1.9 x 10^-4` and `B(t -> cH) < 3.7 x 10^-4` (`deepresearch_may16.txt:1322-1335`). The catalog has T007 for `t -> Hc`, with a current `B(t -> Hc) < 3.4 x 10^-4` (`flavor_catalog/processes/top_higgs_ew/T007.tex:17-33`), and it explicitly keeps only the charm-channel bound because `t -> Hu` correlations are analysis-dependent (`flavor_catalog/processes/top_higgs_ew/T007.tex:63-70`). Plan v1 had T008 for `t -> u h` (`docs/phase_logs/flavor_catalog_plan_v1.md:311-312`), but it is neither in the DA-4 inventory nor the DA-4 deferred list (`flavor_catalog/worklogs/discovery/round_004_convergence.md:13`, `:36`). Judgment: this should not be silently absent. Either add T008 or document that T007 deliberately covers only the charm channel and that T008 is deferred.

4. **LFV kaon modes such as `K_L -> mu e`, `K^+ -> pi^+ mu e`, and `K_L -> pi^0 e mu`.** The external report calls these null tests with effectively zero SM background, secondary for quark-only RS but important for lepton-bulk RS (`deepresearch_may16.txt:431-475`). Plan v1 had K019/K020/K021 (`docs/phase_logs/flavor_catalog_plan_v1.md:237-239`), and DA-4 deferred them as "LFV kaon tails" (`flavor_catalog/worklogs/discovery/round_004_convergence.md:33`). Judgment: the deferral is mostly sound for the 75-process core because the catalog already includes the muon golden channels and mu-e conversion. But if the PI wants the catalog to support quark-lepton unified RS, the external report is right that these should be promoted from tail to at least one representative semileptonic LFV meson row.

5. **LFV B and semileptonic LFV meson decays, for example `B -> K mu e`, `B^0 -> mu tau`, `B^0 -> e tau`.** The external report discusses them in the B table and again in the lepton-flavor section, saying they are subleading to muon golden modes but useful cross-checks in unified quark-lepton warped models (`deepresearch_may16.txt:1060-1118`, `:1700-1753`). Plan v1 had B036/B037 (`docs/phase_logs/flavor_catalog_plan_v1.md:298-299`), and DA-4 deferred them as lower-leverage LFV B modes (`flavor_catalog/worklogs/discovery/round_004_convergence.md:35`). Judgment: sound for quark-only and even for a first lepton-bulk pass, but the catalog should not claim full quark-lepton LFV meson coverage.

6. **Semileptonic asymmetries and CPV in neutral-B mixing: `a_sl^d,s`, `|q/p|_{d,s}`.** The external report lists `Delta Gamma_s`, semileptonic asymmetries, and `|q/p|` as important consistency checks for new `Delta B=2` phases (`deepresearch_may16.txt:579-626`). Our B004 includes `phi_s` and `Delta Gamma_s` (`flavor_catalog/processes/beauty/B004.tex:21-35`), but searches did not reveal a dedicated `a_sl` or `|q/p|_{d,s}` catalog row. Judgment: not a blocker, but a real nuance gap. These are not usually the leading RS bound, yet they are natural companion observables once the catalog has B-mixing phases.

7. **Rare-charm four-body/angular modes such as `D^0 -> pi^+ pi^- mu^+ mu^-` and `D^0 -> K^+ K^- mu^+ mu^-`.** The external report says LHCb angular analyses already exist and calls these future-facing operator-diagnosis tools (`deepresearch_may16.txt:1240-1293`). The catalog has C007 for `D^+ -> pi^+ mu^+ mu^-` (`flavor_catalog/processes/charm/C007.tex:10-18`) and C008 for LFV `D^+ -> pi^+ e mu`, but not the neutral four-body angular program. Judgment: acceptable as deferred tail for v0.1, but the catalog should mention this as the next rare-charm expansion if down-aligned RS becomes a central use case.

8. **Tau/W charged-current universality ratios (`g_tau/g_mu`, `g_tau/g_e`, `g_mu/g_e`).** The external report quotes HFLAV universality ratios and says they become important with non-universal lepton profiles or perturbed charged currents (`deepresearch_may16.txt:1447-1490`). Plan v1 had broad universality rows T022 and L022 (`docs/phase_logs/flavor_catalog_plan_v1.md:326`, `:352-354`), but the final inventory has no tau-universality process and DA-4 does not explicitly defer L021/L022/T022 consistently (`flavor_catalog/worklogs/discovery/round_004_convergence.md:13-15`, `:36-38`). Judgment: this is a small bookkeeping gap. K017 and EW002 cover kaon LFU and CKM unitarity, but not the HFLAV tau-coupling ratios highlighted externally.

## Section 2: Processes / Constraints Where the External Research Disagrees With the Catalog

1. **C004, `D^0 -> mu^+ mu^-`: value/current-best source disagreement.** The external report calls the LHCb `3.1 x 10^-9` 90% CL limit the world's best (`deepresearch_may16.txt:1249-1256`). The catalog records the newer PDG/CMS situation: PDG Live gives `< 2.1 x 10^-9` at 90% CL, CMS gives `< 2.4 x 10^-9` at 95% CL, and LHCb's `3.1 x 10^-9` is correctly retained as the predecessor (`flavor_catalog/processes/charm/C004.tex:21-39`). Type: stale value in external research. Severity: NIT for catalog correctness; no catalog change indicated.

2. **B025/B026, `R_D` and `R_D*`: HFLAV vintage disagreement.** The external report quotes `R(D)=0.342 +/- 0.026`, `R(D*)=0.286 +/- 0.012`, with a `3.14 sigma` tension (`deepresearch_may16.txt:948-960`). The catalog uses the HFLAV CKM 2025 joint average, `R_D=0.358 +/- 0.024`, `R_D*=0.281 +/- 0.011`, correlation `-0.374`, and a combined `3.8 sigma` discrepancy (`flavor_catalog/processes/beauty/B025.tex:19-33`, `flavor_catalog/processes/beauty/B026.tex:21-37`). Type: value/source-date disagreement. Severity: NIT from the catalog side; the catalog is more current and fact-checked.

3. **B009, `B^+ -> tau^+ nu_tau`: measurement-vs-average framing.** The external report quotes Belle II evidence, `[1.24 +/- 0.41(stat) +/- 0.19(syst)] x 10^-4`, as the current status (`deepresearch_may16.txt:925-946`). The catalog uses the HFLAV Dec. 2025 average, `(1.12 +/- 0.19) x 10^-4`, while also listing the Belle II input explicitly (`flavor_catalog/processes/beauty/B009.tex:19-34`). Type: aggregation interpretation. Severity: NIT; the external report is not wrong about the Belle II input, but it is not the best catalog headline.

4. **T007, `t -> Hc`: limit freshness disagreement.** The external report quotes `B(t -> cH) < 3.7 x 10^-4` (`deepresearch_may16.txt:1329-1335`). The catalog records a PDG 2026/ATLAS-combination headline of `< 3.4 x 10^-4`, with CMS comparison `< 0.037%` (`flavor_catalog/processes/top_higgs_ew/T007.tex:17-37`). Type: value/source edition. Severity: NIT; the catalog's current value is tighter.

5. **B018/B019, `R_K` and `R_K*`: bin-label ambiguity.** The external report quotes `R_K=0.947 +/- 0.047` and `R_K*=1.028 +/- 0.074` while describing a "standard low-q^2 region" (`deepresearch_may16.txt:807-820`). The catalog separates the bins: B018 central-bin `R_K=0.947 +/- 0.047` for `1.1<q^2<6.0`, while B019 has low-bin `R_K*=0.927 +/- 0.097` and central-bin `R_K*=1.028 +/- 0.074` (`flavor_catalog/processes/beauty/B018.tex:20-40`, `flavor_catalog/processes/beauty/B019.tex:20-38`). Type: validity-range/binning ambiguity. Severity: WARNING only if the external text is used as prose; the catalog itself is precise.

I found no BLOCKER where a fact-checked catalog value is contradicted by a stronger, better-sourced external value. The disagreements are mostly stale or less precise external summaries.

## Section 3: Subtleties / Nuances the External Research Surfaces That We Did Not Make Explicit

1. **Custodial protection reshuffles the hierarchy.** The external report says custodial RS partially protects `Z b_L b_L`, making rare semileptonic modes and B-mixing comparatively more important (`deepresearch_may16.txt:36-42`). T010 already says custodial embeddings determine whether `Zbb` is leading or diagnostic (`flavor_catalog/processes/top_higgs_ew/T010.tex:31-40`), but B015-B019 and B022/B023 could say it more explicitly. Suggested wording: "In custodial RS, the reduced `Z b_L b_L` pressure can make semileptonic `b -> s` channels and B-mixing phases relatively more discriminating than in minimal bulk RS."

2. **Down-sector alignment promotes up-sector observables from important to mandatory.** The external report repeats this in the model-class table and baseline recommendations (`deepresearch_may16.txt:64-66`, `:1492-1494`, `:1884-1886`). C001 notes the up-sector complement (`flavor_catalog/processes/charm/C001.tex:43-49`), but the top entries should carry the same triage language. Suggested wording for C001-C004 and T001/T002/T005/T006/T007: "If the model invokes down-sector alignment to evade kaon and B constraints, this observable should be promoted from diagnostic to mandatory."

3. **CP protection must be tested across EDMs and penguin phases, not just `epsilon_K`.** The external report names EDMs, `epsilon_K`, `S_phi K_S`, and `B_s` penguin phases as the stress test for CP-aligned RS (`deepresearch_may16.txt:78-81`). The catalog has E001/E004 and B033/B034, but the cross-entry synthesis is not obvious. Suggested wording for E001/E004/B033/B034/K001/K003: "A CP-aligned RS claim should be checked jointly against EDMs and CP-violating kaon/B-penguin observables, because suppressing one sector does not guarantee phase alignment in the others."

4. **Exclusive radiative B modes are not just lower-statistics copies of the inclusive rate.** The external report emphasizes their sensitivity to right-handed dipoles (`deepresearch_may16.txt:702-719`). B011 says inclusive is separated from exclusive modes (`flavor_catalog/processes/beauty/B011.tex:13-19`), but does not spell out what is lost. Suggested wording for B011 or a future B012/B013: "The inclusive branching fraction mainly constrains `|C_7|`-like combinations; time-dependent or helicity observables in exclusive radiative modes are the cleaner probes of right-handed dipoles."

5. **Meson LFV is a quark-lepton linkage test.** The external report frames LFV kaon/B decays as cross-checks once quark and lepton flavor symmetries are tied together (`deepresearch_may16.txt:458-475`, `:1744-1751`). Suggested wording for the charged-lepton family overview or deferred-scope note: "Semileptonic LFV meson decays are secondary to muon golden modes for generic sensitivity, but they become diagnostic in unified quark-lepton warped flavor models."

## Section 4: Overall Assessment of the External Research Itself

The external research gets the big physics map right. It correctly emphasizes that a modern warped-flavor inventory cannot stop at the old `Delta F=2` set: kaons, B mixing, dipoles, semileptonic FCNCs, charm/top for alignment, charged LFV, and EDMs all matter (`deepresearch_may16.txt:3-8`, `:1917-1925`). This aligns with the 75-process catalog inventory (`flavor_catalog/worklogs/discovery/round_004_convergence.md:8-15`) and with Opus's conclusion that the catalog keeps RS relevance conservative rather than overstating hard exclusions (`flavor_catalog/signoff/round_001_index.md:117-120`).

It also gets several qualitative priorities right that our catalog already reflects: `epsilon_K` as the classic anarchic-RS pressure point, rare kaons as clean `Delta S=1` tests, `B -> X_s gamma` as a dipole anchor, `R_K/R_K*` no longer being anomaly-driven, `B_s -> mu mu` as a clean leptonic rare decay, and electron/neutron EDMs as CP-protection tests. Representative catalog entries make the same points in K001, K004/K005, B011, B018/B019, B005, E001, and E004.

Where the external research is weaker, the catalog is generally better. Several "current" numerical statements are stale or less precise than our fact-checked entries: C004, B009, B025/B026, T007, and the `R_K*` bin label. The external reference block is also visibly generated from web snippets, including `utm_source=chatgpt.com` URLs and PDF extraction artifacts (`deepresearch_may16.txt:1937-2088`), whereas the catalog has local snapshots, sidecars, and independent fact-check reports.

Where the external research caught real catalog weaknesses, they are mostly scope/accounting issues: missing top photon FCNCs, missing `t -> uH`, no explicit `a_sl^{d,s}` row, no tau charged-current universality row, and overly terse deferral of radiative-helicity B observables. The strongest of these are T003/T004/T008 because they were already in plan v1 but are absent from both the 75-process inventory and the DA-4 deferred list.

Both the external report and the catalog still have gaps at the implementation level. Neither is a likelihood backend. Both repeatedly depend on future model choices for `Delta F=1` matching, angular observables, nonlocal charm, nonleptonic penguins, EDM operator running, nuclear overlap integrals, and correlations across observables. The catalog is much better as a source-controlled, fact-checked inventory; the external report is useful as an independent triage sanity check and as a prompt to tighten deferred-scope bookkeeping.
