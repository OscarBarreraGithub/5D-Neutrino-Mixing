# PKA Worklog: K004

Process assignment: `K^+ -> pi^+ nu nubar`, kaon family.  I matched this
unambiguously to plan v1 Section C row `K004`, which lists the process as the
ultra-rare `s -> d nu nubar` charged kaon mode with initial coverage
`NOT-SEEN`.  I used family `kaon` because the row is in the kaon-sector table.

Required readings completed before drafting: `flavor_catalog_plan_v1.md`,
`flavor_catalog_orchestrator_decisions.md`, the Opus signoff, the process
template, and the catalog README.  I followed the orchestrator's source policy:
minimal text snapshots only, no publisher PDFs.  The main literature checked was
the latest NA62 arXiv update `2604.12649`, the published NA62 observation paper
`2412.12015` / JHEP 02 (2025) 191, Buras--Venturini `2203.10099`, Brod--
Gorbahn--Stamou `2105.02868`, and the requested CFW-era baseline
`0804.1954`.  I also searched for a current PDG/PDGLive value, but the most
actionable and newest value for this process is the NA62 2026 preliminary
combination, so I treated it as the `pdg_or_equivalent` experimental value.

Conclusion: the current experimental-equivalent number is the NA62 2016--2024
combination, `B(K+ -> pi+ nu nubar) = (9.6^{+1.9}_{-1.8}) x 10^{-11}`.  The
published observation anchor remains the 2016--2022 NA62 result,
`(13.0^{+3.3}_{-3.0}) x 10^{-11}`.  For SM comparison I recorded two modern
benchmarks: Buras--Venturini `(8.60 +/- 0.42) x 10^{-11}` and
Brod--Gorbahn--Stamou `7.73(61) x 10^{-11}`.  The former is more precise but
comes with the stated no-new-physics assumption in the Delta F = 2 inputs.

Coverage greps were run across `quarkConstraints/`, `flavorConstraints/`,
`scanParams/`, `neutrinos/`, `yukawa/`, `qcd/`, `scripts/`, `tests/`, and
`docs/`.  Specific searches for `K+`, `pi+`, `nu nubar`, `NA62`, and `KOTO`
returned only catalog planning text, while code inspection showed the live
quark constraint surface is Delta F = 2 mixing (`epsilon_K`, `K`, `B_d`, `B_s`,
`D0`) plus unrelated lepton LFV.  I therefore marked code coverage `NO`.
