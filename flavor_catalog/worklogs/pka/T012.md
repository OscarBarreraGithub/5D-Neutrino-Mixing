# PKA worklog: T012

Assigned process: `Z -> c cbar` pole observables, explicitly including
`R_c`, `A_FB^{0,c}`, and `A_c`, using process id `T012`.

I first checked out `flavor-catalog/2026q2` and read the required plan,
orchestrator decisions, Opus signoff, process template, and catalog README in
the requested order.  Plan v1 splits this material across T012 (`R_c`) and T013
(`A_FB^c`, `A_c`), but the dispatch instruction requested one combined charm
Z-pole entry under T012.  I recorded T013 as a related process and left an
open issue for the WA/orchestrator to mark it covered or request a later split.

For canonical values I downloaded the PDG 2025 Z-boson listing to `/tmp`,
converted it with `pdftotext`, and extracted only a small normalized text
snapshot.  The current PDG values captured are `R_c^0 = 0.1721 +/- 0.0030`,
`A_FB^{0,c} = 0.0707 +/- 0.0035`, and `A_c = 0.670 +/- 0.027`; the PDG listing
prints `A_FB^{0,c}` as `7.07 +/- 0.35%`, which I converted to the dimensionless
form.  I also downloaded and checked the LEP/SLC final Z-resonance combination,
`hep-ex/0509008`, because it is the final heavy-flavour source behind the PDG
listing.  No publisher PDFs were stored.

For RS relevance and post-2008 context I consulted arXiv `0804.1954`,
`0807.4937`, `1401.2447`, and `2107.00616`.  The conclusion is that T012 is a
precision electroweak coupling constraint rather than a flavor-changing decay:
important for warped fermion localization and Z-coupling shifts, but highly
model-dependent once translated into 5D parameters.
