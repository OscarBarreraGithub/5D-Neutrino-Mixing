# PKA Worklog: E007 Radium/Xenon EDMs

I followed the required catalog plan on branch `flavor-catalog/2026q2` and
worked only under `flavor_catalog/`.  The assigned process is the pair of
radium-225 and xenon-129 atomic EDM searches, process id `E007` in family
`edm_neutrino`.  I treated the direct experiment papers as PDG-equivalent
sources because I did not find a standalone PDG Live datablock for Ra/Xe
atom EDMs during this PKA pass.

Searches used included `225Ra EDM limit Bishof 2016`, `129Xe EDM Sachdeva
2019`, `PDG 129Xe electric dipole moment`, `Argonne radium EDM`, `Heidelberg
129Xe EDM upgrade`, and `KU Leuven RaF AcF EDM Schiff moment`.  I saved
minimal text snapshots only: arXiv abstract pages for the Ra and Xe
measurements, the Argonne Ra EDM program page, the DFG Heidelberg upgrade
proposal, the KU Leuven RaF/AcF project page, and arXiv:0804.1954 as the
2008 warped-flavor baseline.  I did not cache publisher PDFs.

The canonical values recorded are `|d(225Ra)| < 1.4e-23 e cm` at 95% CL from
Bishof et al. 2016 and `|d_A(129Xe)| < 1.4e-27 e cm` at 95% CL from
Sachdeva et al. 2019.  I also recorded the 2015 first Ra result
`5.0e-22 e cm`, the independent Heidelberg-centered Xe result
`1.5e-27 e cm`, and proposal-level sensitivity statements from Argonne, DFG,
and KU Leuven.  These proposal statements are kept in `post_2008_context`,
not in `pdg_or_equivalent`, except where they report measured experimental
observables.

For code coverage I ran the three required catalog greps over
`quarkConstraints`, `qcd`, `flavorConstraints`, `neutrinos`, `yukawa`,
`warpConfig`, `solvers`, `scanParams`, and `tests`, plus a focused search for
Ra, radium, Xe, xenon, Schiff, octupole, diamagnetic, chromo-EDM, Weinberg,
and generic EDM/dipole terms.  No Ra or Xe EDM implementation was found.  The
only adjacent code is the existing off-diagonal `mu -> e gamma` dipole helper
in `flavorConstraints/muToEGamma.py`, so I marked code coverage `NO` and
implementation difficulty `HIGH`.
