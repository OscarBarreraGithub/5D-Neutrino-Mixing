template, and catalog README in order after checking out

baseline.  Queries included `PDG epsilon_K value 2.228 0.011 10^-3`,
`Brod Gorbahn Stamou epsilon_K 2.161`, `FLAG Review 2024 B_K`, and

policy.  I used arXiv abstracts and source extracts when exact values were
needed.
