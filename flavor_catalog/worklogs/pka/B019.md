# PKA Worklog: B019

Assigned process: `R_K*` in `B0 -> K*0 ell+ ell-`, beauty family, process ID
`B019`. I created only B019 paths under `flavor_catalog/` and did not touch
catalog indexes, the process template, or neighboring deposits.

Source work started from the B018 companion entry and direct HFLAV rare-decay
tables. Searches and checks included `HFLAV R_K* Dec 2025 B0 K*0 mu mu e e`,
`site:hflav-eos.web.cern.ch Dec2025 R_K* K*0`, `LHCb 2212.09153 R_K* 0.927
1.027`, and `Bordone Isidori Pattori R_K* SM 0.983`. The HFLAV Dec. 2025 RBR
index exposes the neutral `K*(892)0` ratio pages. I used the `0.1 < q^2 < 1.1`
page as the current LHCb-bin low-q2 value, `0.927 +/- 0.097`, and the
`1.1 < q^2 < 6.0` page as the central-bin average, `1.028 +/- 0.074`.

Primary experimental context is LHCb arXiv `2212.09153`, whose source TeX gives
the superseding low- and central-q2 `R_K*` values and states compatibility with
the SM at `0.2` standard deviations for the four LFU measurements. I also kept
LHCb arXiv `1705.05802`, because it is the post-2008 anomaly milestone with
`0.66` and `0.69` central values and roughly `2.1--2.5 sigma` tensions. Theory
context is Bordone--Isidori--Pattori `1605.07633`, especially the QED-sensitive
low-q2 SM prediction and central-bin unity expectation. CFW `0804.1954` is
included only as the plan-required RS-flavor baseline.

Coverage greps were run over `quarkConstraints/`, `qcd/`, `flavorConstraints/`,
`neutrinos/`, `yukawa/`, `warpConfig/`, `solvers/`, `scanParams/`, and
`tests/`. Targeted searches for `R_K*`, `K*0 ee/mumu`, `C_9`, `C_10`, LFU, and
`b -> s ell ell` returned no matches. Existing code covers Delta F = 2 neutral
mixing and `mu -> e gamma`, so I marked coverage `NO` and implementation
difficulty `HIGH`.
