# PKA Worklog: T018

Process: `h -> mu tau`, assigned as `T018` in the `top_higgs_ew` family.
The plan row specified ATLAS/CMS Higgs LFV, the historical CMS 2015 hint, and
current limits at the `10^-3` level.

I read the required plan, orchestrator decisions, Opus signoff, process
template, and catalog README before writing files.  For the canonical value I
used the PDG 2025 Higgs review section on flavor-violating Higgs decays, then
checked it against the primary ATLAS and CMS papers.  The current strongest
single-experiment limit is the CMS full Run-2 result,
`B(H -> mu tau) < 0.15%` observed and expected at 95% CL, from
`137 fb^-1` at `sqrt(s)=13 TeV` (arXiv:2105.03007).  The latest ATLAS full
Run-2 result is `B(H -> mu tau) < 0.18%` observed and `0.09%` expected at
95% CL from `138 fb^-1` (arXiv:2302.05225).  I also captured the CMS 2015
paper (arXiv:1502.07400), since it reported the `2.4 sigma` excess,
`p=0.010`, best fit `(0.84 +0.39 -0.37)%`, and limit `<1.51%`.

Search queries included "PDG Higgs boson lepton flavor violating h -> mu tau
branching fraction limit", "CMS latest H to mu tau lepton flavor violating
Higgs decay branching fraction upper limit", "ATLAS H -> mu tau 138 fb 0.18",
and "Harnik Kopp Zupan lepton flavor violating Higgs decays".  I rejected raw
CMS and ATLAS public-result HTML pages because the raw pages exceeded the
requested sub-100 KB snapshot target; the arXiv abstract snapshots contain the
needed numerical values.  I did not track any publisher PDFs.  I included
Harnik-Kopp-Zupan arXiv:1209.1397 for Higgs-LFV model context, CFW
arXiv:0804.1954 as the required 2008 baseline, and Perez-Randall
arXiv:0805.4652 because the current repo lepton-sector LFV implementation is
based on that lane.
