# PKA Worklog: C007

Assigned process: `C007`, charm family, \(D^+ \to \pi^+\mu^+\mu^-\).  I
checked out `flavor-catalog/2026q2` first, then read the planner v1 document,
orchestrator decisions, Opus signoff, the process template, and the catalog
README in the requested order.  The plan row identifies this as a rare
\(c\to u\ell\ell\) semileptonic FCNC probe with expected `NOT-SEEN` code
coverage.

Source extraction found that the canonical current value is newer than the
older LHCb 2013 limit often associated with this channel.  PDG Live/API
S031.42, accessed on 2026-05-16, lists the current used value
\(\mathcal B(D^+\to\pi^+\mu^+\mu^-)<6.7\times10^{-8}\) at 90% CL from
LHCb JHEP 06 (2021) 044 / arXiv:2011.00217.  The LHCb source table gives the
same 90% CL limit and a 95% CL companion limit \(<7.4\times10^{-8}\), based on
1.6 fb\(^{-1}\) of 2016 \(pp\) data.  I retained LHCb arXiv:1304.6365 as the
predecessor nonresonant search: \(<7.3(8.3)\times10^{-8}\) at 90% (95%) CL,
using 1.0 fb\(^{-1}\) at 7 TeV.

For theory and RS context I saved de Boer and Hiller, arXiv:1510.00311, because
it is directly about \(D_{(s)}\to P\ell^+\ell^-\) rare charm and
Wilson-coefficient interpretation outside resonance regions.  I also saved a
minimal CFW arXiv:0804.1954 snapshot for the required post-2008 RS-flavor
baseline.  Licensing policy was followed: no publisher PDFs are stored, and all
source files are short text snapshots with URL, access date, and sha256 in the
manifest.

For code coverage I ran the three required catalog greps over
`quarkConstraints`, `qcd`, `flavorConstraints`, `neutrinos`, `yukawa`,
`warpConfig`, `solvers`, `scanParams`, `tests`, `docs`, and `scripts`, plus a
focused implementation-dir search for \(D^+\pi^+\mu^+\mu^-\), rare charm,
\(c\to u\), and \(C_{9,10}\) patterns.  Only D0 mixing and the unrelated
\(\mu\to e\gamma\) check were found, so C007 is marked `NO`.  Main unresolved
item for WA/CA: confirm the exact LHCb 2021 resonance/phase-space definition
before any future hard recast.
