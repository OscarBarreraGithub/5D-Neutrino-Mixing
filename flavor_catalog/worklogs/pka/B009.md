# PKA Worklog: B009

Process chosen: `B009`, `B^+ -> tau^+ nu_tau`, from plan v1 Section C,
beauty-sector row "Charged-Higgs/charged-current sensitive seed; HFLAV/PDG."
The family mapping is therefore `beauty`.

I first read the required plan, orchestrator decisions, Opus signoff,
template, and catalog README, then inspected the existing untracked `B009`
draft and verified it rather than overwriting it.  For the canonical value I
used the HFLAV Rare B Decays
End-of-December-2025 page for `B^+ -> tau^+ nu_tau`, because it is newer than
the PDG 2025 average and explicitly includes the Belle II 2025 input.  I also
queried the PDG 2025 REST API for `S041.184`; it gives `(1.09+-0.24)E-4` with
scale factor information and lists the Belle/BaBar inputs used in the PDG
average.  For the SM comparison I used UTfit Summer 2024, which quotes the
`B(B->tau nu) * 10^4` SM prediction as `0.865 +- 0.041`.

Primary experimental snapshots were taken from arXiv abstract pages for Belle
2013 hadronic tag (`1208.4678`), Belle 2015 semileptonic tag (`1503.05613`),
Belle II 2025 hadronic tag (`2502.04885`), BaBar 2013 hadronic tag
(`1207.0698`), and BaBar 2010 semileptonic tag (`0912.2453`).  No publisher
PDFs were downloaded or committed.  I also saved a small arXiv abstract
snapshot for Csaki--Falkowski--Weiler (`0804.1954`) only as the requested
2008-era baseline.
