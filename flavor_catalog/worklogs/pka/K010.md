required plan, orchestrator-decision, Opus signoff, template, and catalog README
files after checking out `flavor-catalog/2026q2`. The key policy constraints
were the minimal-text-snapshot licensing rule, status-history-only sidecars,
and the requirement that measured observables live in `pdg_or_equivalent` while
theory normalizations remain separate.

Source search focused on PDG/NA48 and the `K_L -> pi0 e+ e-` companion process.
Useful queries included `PDG K_S -> pi0 e+ e- branching ratio NA48 5.8 10^-9`
and `pdgLive K_S pi0 e+ e- S012`. The canonical source is PDG pdgLive API
listing `S012.10`, which reports the NA48/1 measurement
`BR(K0S -> pi0 e+ e-, m(ee)>0.165 GeV) = (3.0+1.5-1.2+-0.2)E-9` and the PDG
footnote extrapolation `5.8+2.9-2.4E-9`. I also saved the arXiv abstract-page
snapshot for NA48/1 `hep-ex/0309075`, which independently records the 7 observed
events, 0.15 expected background, the partial branching fraction, and the
full-rate extrapolation. I did not track any publisher PDFs.

For context I kept minimal arXiv snapshots for Isidori--Smith--Unterdorfer

the 2008 RS-flavor baseline `0804.1954`. These are not used as measured K010
values; they document the indirect-CP role of the `K_S` input and the
post-2008 emphasis on `K_{L,S}->pi0 l+l-` branching ratios and spectra.
