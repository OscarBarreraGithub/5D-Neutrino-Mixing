# PKA Worklog: T004

Process: `t -> u gamma` top FCNC photon decay.  I first checked out
`flavor-catalog/2026q2` and read the required plan, orchestrator decisions,
Opus signoff, LaTeX process template, and catalog README.  The key execution
points for this process were the no-publisher-PDF rule, the status-history
schema, the code-coverage grep requirement, and the Opus arbitration note that
measured observables belong in `pdg_or_equivalent` while theory-normalization
context should stay separate.

Source search queries included `PDG t -> u gamma branching fraction limit`,
`B(t -> u gamma) 0.85e-5`, `Search for flavour-changing neutral-current
couplings between the top quark and the photon ATLAS`, and `CMS TOP-21-013
t u gamma`.  The canonical sources kept were PDG pdgLive 2026 for the combined
`t -> gamma q` row, ATLAS arXiv:2205.02537 for the tight flavor-separated
`t -> u gamma` chirality benchmarks, and the CMS-TOP-21-013 public page for the
newer CMS result.  I also saved minimal context snapshots for arXiv:0804.1954
as the post-2008 baseline and arXiv:1709.03975 for SM/projection context.
ResearchGate, institutional mirror PDFs, and publisher PDFs were rejected
because the catalog policy asks for minimal text snapshots and primary or
canonical pages.

The main source nuance is that PDG lists `t -> gamma q` with `q = u,c`, whereas
T004 needs the flavor-separated `t -> u gamma` constraint.  I therefore put the
PDG combined limit and ATLAS/CMS mode-resolved results in the sidecar, with the
ATLAS left-handed benchmark as the headline limit.  Theory and projection
numbers were kept in `paper_era_reference`.

For code coverage, I reran the required catalog greps across
`quarkConstraints/`, `qcd/`, `flavorConstraints/`, `neutrinos/`, `yukawa/`,
`warpConfig/`, `solvers/`, `scanParams/`, and `tests/`, followed by a targeted
`t -> u gamma` / `tu gamma` / `tq gamma` search.  No implementation was found;
only the known neutral-meson mixing and `mu -> e gamma` paths appeared.
