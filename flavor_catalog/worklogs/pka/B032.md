# PKA Worklog: B032

Assigned process: `B032`, beauty family, `Bbar -> pi Kbar` nonleptonic
charmless decays.  I read the required plan, orchestrator decisions, Opus
signoff, process template, and catalog README before drafting.

I used HFLAV as the canonical source for the four current `B -> K pi`
branching fractions and direct CP asymmetries.  The active HFLAV Rare B page
marks End of December 2025 as the latest update, so I used the Dec2025
individual HTML pages rather than the May2025 PDF result initially found by
search.  The extracted HFLAV values are stored in
`flavor_catalog/references/B032/hflav_dec2025_btopik_values.txt`.  For the
neutral time-dependent CP parameters I used the PDG 2025 REST API entries
`S042CKP` and `S042SKP`, stored in
`pdg2025_k0pi0_time_dependent_cp_api.txt`.  I recorded both the HFLAV
time-integrated direct asymmetry for `B0 -> K0 pi0` and the PDG time-dependent
`C` coefficient because the sign convention differs between Belle-style
`A` and PDG/HFLAV `C`.

Post-2008 context was taken from small arXiv text snapshots: Belle II 2024
for the full `B -> K pi` branching/direct-CP update and sum-rule test, LHCb
2021 for the sharpened `K pi` direct-asymmetry puzzle in `B+ -> K+ pi0`,
Belle II 2023 for `B0 -> K_S pi0` time-dependent CP, and arXiv:0804.1954 as
the RS-flavor benchmark era reference.  I stored no publisher PDFs and no full
PDG/HFLAV PDFs.

For code coverage I ran the requested implementation-directory greps across
`quarkConstraints`, `qcd`, `flavorConstraints`, `neutrinos`, `yukawa`,
`warpConfig`, `solvers`, `scanParams`, and `tests`.  A focused `B -> K pi` /
charmless / nonleptonic / electroweak-penguin grep returned no hits.  Broader
greps found only the existing neutral-meson Delta F = 2 surface and the
lepton `mu -> e gamma` module.  I therefore marked code coverage `NO` and
implementation difficulty `HIGH`.
