Title: Search for the Decays B^0_(s) -> e+mu- and B^0_(s) -> e+e- in CDF Run II
Authors: CDF Collaboration, T. Aaltonen et al.

Minimal extract for B007.

Abstract/context:
  The CDF Run II search covers lepton-flavor-violating B^0_(s) -> e+mu-
  and flavor-changing neutral-current B^0_(s) -> e+e- decays.  It used
  2 fb^-1 of p pbar collisions at sqrt(s) = 1.96 TeV collected with CDF II.
  The observed number of B^0_(s) candidates was consistent with background.

Exact electronic limits used for B007:
  The arXiv abstract metadata repeats the e+mu limit text where the e+e-
  limits should appear.  The exact B^0_(s) -> e+e- numerical values are
  taken from the PDG API extract and from the LHCb 2020 paper's reference
  to this CDF result:
    B(Bs0 -> e+e-) < 2.8 x 10^-7 at 90% CL.
    B(B0 -> e+e-) < 8.3 x 10^-8 at 90% CL.
