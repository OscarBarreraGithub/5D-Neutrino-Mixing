arXiv: 2103.12994 [hep-ex], version v6
Title: Search for lepton-flavor-violating tau-lepton decays to l gamma at Belle
Authors: K. Uno et al. (Belle Collaboration)
Journal: JHEP 10 (2021) 019
DOI: 10.1007/JHEP10(2021)019
Submitted: 2021-03-24; last revised: 2021-09-06

Relevant excerpt:

An updated search via tau+- -> l+- gamma using full data of the Belle
experiment, corresponding to an integrated luminosity of 988 fb^-1, is
reported for charged lepton flavor violation. No significant excess over
background predictions from the Standard Model is observed. Upper limits on
the branching fractions are set at 90% confidence level:

  B(tau+- -> mu+- gamma) <= 4.2 x 10^-8
  B(tau+- -> e+- gamma)  <= 5.6 x 10^-8

Extraction note:
This is the main post-2008 Belle full-data update for tau -> l gamma. For
the electron mode it does not supersede the stronger BaBar/PDG limit.
