Title: Differential branching fraction and angular analysis of Lambda_b^0 -> Lambda mu+ mu- decays
Local snapshot type: minimal arXiv metadata and value extract; no publisher PDF stored.

Key extracted values from the arXiv abstract:
  Dataset: pp collision data collected by LHCb, integrated luminosity 3.0 fb^-1.
  Evidence of signal was observed below the square of the J/psi mass.
  For 15 < q^2 < 20 GeV^2/c^4:
    dB(Lambda_b^0 -> Lambda mu+ mu-) / dq^2
      = (1.18 +0.09 -0.08 +-0.03 +-0.27) x 10^-7 (GeV^2/c^4)^-1
    uncertainties are statistical, systematic, and due to the normalization
    mode Lambda_b^0 -> J/psi Lambda.
  Angular observables for 15 < q^2 < 20 GeV^2/c^4:
    A_FB^l = -0.05 +- 0.09(stat) +- 0.03(syst)
    A_FB^h = -0.29 +- 0.07(stat) +- 0.03(syst)

Catalog note:
  This is the primary modern LHCb source for the q^2-binned differential
  branching fraction and first angular measurements used in the B021 draft.
