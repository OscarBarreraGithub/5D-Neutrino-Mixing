Title: Search for flavour-changing neutral-current couplings between the top
quark and the Higgs boson in multi-lepton final states in 13 TeV pp collisions
with the ATLAS detector.
Authors: ATLAS Collaboration.
Submitted: 2024-04-02; last revised: 2024-12-04.
Journal reference: Eur. Phys. J. C 84 (2024) 757.
Related DOI: 10.1140/epjc/s10052-024-12994-1.

Relevant abstract excerpt:
The analysis uses 140 fb^-1 of pp data at sqrt(s) = 13 TeV. It considers ttbar
production with one top decaying via t -> H q and associated Ht production,
where q = u,c.

The multilepton-only observed (expected) 95% CL limits are
B(t -> H u) < 2.8 (3.0) x 10^-4 and
B(t -> H c) < 3.3 (3.8) x 10^-4.

Combining with other ATLAS tHq FCNC searches using H -> b b, H -> gamma gamma,
and H -> tau+ tau- gives observed (expected) 95% CL limits
B(t -> H u) < 2.6 (1.8) x 10^-4 and
B(t -> H c) < 3.4 (2.3) x 10^-4.
