Relevant extracted values from Table 10.8:
Global fit of oblique parameters to electroweak precision observables.

Fit with U fixed to zero:
S = 0.026 +/- 0.075
T = 0.047 +/- 0.066
U = 0
Correlation rho(S,T) = 0.90

Fit with S, T, and U floating:
S = 0.021 +/- 0.096
T = 0.04 +/- 0.12
U = 0.008 +/- 0.092
Correlations: rho(S,T) = 0.91, rho(S,U) = -0.62, rho(T,U) = -0.83.

Relevant extracted one-parameter 95% probability bounds:
T = U = 0: -0.088 <= S <= 0.045; if S >= 0, S <= 0.054.
S = U = 0: -0.032 <= T <= 0.084; if T >= 0, T <= 0.078.
S = T = 0: -0.047 <= U <= 0.120; if U >= 0, U <= 0.110.

Relevant RS/composite context from the same review:
For warped extra dimensions, sizeable corrections to S are generated by mixing
between Standard Model gauge bosons and KK excitations, with
S approximately 30 v^2 / M_KK^2. From Table 10.8 the review quotes S <= 0.18
at 95% probability, corresponding to M_KK greater than about 3.2 TeV for the
KK gauge-boson mass scale.
