Local snapshot purpose: Minimal extract for B033, B0 -> phi K0 time-dependent CP asymmetry.

Context from the HFLAV page:
HFLAV lists B_d -> phi K_S under "Time-dependent CP Asymmetries in b -> qq-bar s
(penguin) Transitions".  The page states that in the Standard Model the
b -> s penguin transition carries approximately the same weak phase as the
b -> c c-bar s amplitude used to obtain sin(2 beta), and that new virtual
particles may generate deviations from sin(2 beta_eff)(b -> qq-bar s) ~
sin(2 beta)(b -> c c-bar s).

HFLAV convention:
S = -eta_CP * sin(2 beta_eff), C = -A.  The same HFLAV page later states
eta_CP = -1 for phi K_S, eta' K_S, etc., so the listed sin(2 beta_eff)
equals the usual S coefficient for phi K_S under this convention.
