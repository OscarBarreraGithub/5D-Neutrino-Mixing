Convention note for catalog prose:
For the time-dependent B0 -> K0 pi0 CP eigenstate, the coefficient C is the
direct-CP coefficient in the usual time-dependent notation; many Belle papers
quote A = -C.  The sidecar records C and S explicitly rather than silently
converting conventions.
