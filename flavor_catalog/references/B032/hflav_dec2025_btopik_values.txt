Branching-fraction averages from the individual HFLAV HTML pages.
Units are 10^-5:
- B(B+ -> K0 pi+) = 2.392 +/- 0.062. HFLAV table p=0.70, ndf=5.
  The same row lists the PDG value 2.389 +/- 0.063.
- B(B+ -> K+ pi0) = 1.322 +/- 0.043. HFLAV table p=0.62, ndf=3.
  The same row lists the PDG value 1.323 +/- 0.043.
- B(B0 -> K+ pi-) = 2.007 +/- 0.040. HFLAV table p=0.80, ndf=17.
  The same row lists the PDG value 2.003 +/- 0.041.
- B(B0 -> K0 pi0) = 1.012 +/- 0.043. HFLAV table p=0.70, ndf=3.
  The same row lists the PDG value 1.012 +/- 0.043.

Direct CP-asymmetry averages from the individual HFLAV HTML pages.
Units are 10^-2:
- A_CP(B+ -> K0_S pi+) = -2.67 +/- 0.87. HFLAV table p=0.77, ndf=4.
  The update includes an unpublished LHCb input shown in blue: -2.8 +/- 0.5 +/- 0.9.
- A_CP(B+ -> K+ pi0) = +2.7 +/- 1.2. HFLAV table p=0.62, ndf=4.
  The LHCb input is +2.5 +/- 1.5 +/- 0.7.
- A_CP(B0 -> K+ pi-) = -8.31 +/- 0.31. HFLAV table p=0.54, ndf=4.
  The LHCb input is -8.31 +/- 0.34 and combines Run 2 with Run 1.
- A_CP(B0 -> K0 pi0) = -1 +/- 13. HFLAV table includes the Belle II input
  -1 +/- 12 +/- 4, described as a combination of time-integrated and
  time-dependent analyses using BLUE.
