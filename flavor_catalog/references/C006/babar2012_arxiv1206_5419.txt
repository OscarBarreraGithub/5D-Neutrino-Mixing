Title: Search for the decay modes D0 -> e+e-, D0 -> mu+mu-, and D0 -> e mu
Authors: BABAR Collaboration
Journal reference: Phys. Rev. D 86, 032001 (2012)
DOI: 10.1103/PhysRevD.86.032001
Submitted: 2012-06-23
Last revised: 2012-08-01

Extracted abstract facts:
  BABAR searched for D0 -> e+e-, D0 -> mu+mu-, and D0 -> e mu in continuum
  e+e- -> c cbar events recorded in a data sample corresponding to an
  integrated luminosity of 468 fb^-1.
  Observed event yields were consistent with expected backgrounds, apart from
  a D0 -> mu+mu- upward fluctuation at the 5% level.
  Using the Feldman-Cousins method, BABAR set 90% confidence-level intervals:
    B(D0 -> e+e-) < 1.7 x 10^-7,
    B(D0 -> mu+mu-) within [0.6, 8.1] x 10^-7,
    B(D0 -> e mu) < 3.3 x 10^-7.
