Minimal extracted value:
The final article metadata abstract records an upper limit for the spontaneous
muonium-to-antimuonium conversion probability
P_M Mbar <= 8.3 x 10^-11 at 90% C.L. in a 0.1 T magnetic field.

ArXiv metadata note:
The arXiv abstract for hep-ex/9807011 records the same MACS/PSI result but
uses 8.2 x 10^-11. For catalog numerics, use the PDG S004MC value and footnote
as canonical: P_M Mbar < 8.3 x 10^-11 and G_C/G_F < 0.0030, both at 90% CL.
