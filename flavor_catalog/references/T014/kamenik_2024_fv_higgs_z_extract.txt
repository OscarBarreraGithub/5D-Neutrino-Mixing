Minimal local text extract for T014.

Abstract extract:
The paper assesses the FCC-ee reach for Z/h -> bs, cu decays as a function of
jet-tagging performance and updates SM predictions for the corresponding
branching ratios and indirect constraints on flavor-violating Higgs and Z
couplings to quarks.

Table I extract:

Decay        SM prediction          exp. bound       indirect constraint
B(Z -> bs)  (4.2 +/- 0.7) x 10^-8   2.9 x 10^-3      6 x 10^-8
B(Z -> bd)  (1.8 +/- 0.3) x 10^-9   2.9 x 10^-3      6 x 10^-8
B(Z -> cu)  (1.4 +/- 0.2) x 10^-18  2.9 x 10^-3      4 x 10^-7

The table caption states that these are SM predictions and current
experimental upper limits from direct searches or indirect constraints.

Direct-limit text extract:
Existing direct limits on non-standard hadronic decays of the Z follow from
the agreement of the measurement and SM prediction for the Z hadronic width,
giving B(Z -> qq') < 2.9 x 10^-3 at 95% CL.

FCC-ee reach text extract:
For Z -> bs the paper states that the expected 95% CL bound is O(10^-6) with
1% systematics and O(10^-7) with aggressive 0.1% systematics, still above the
SM value. The supplementary table gives example 95% upper limits:

  (TPR,FPR,syst.) = (0.4,10^-4,1%)   -> B(Z -> bs) < 1.8 x 10^-6
  (0.4,10^-4,0.1%)                  -> B(Z -> bs) < 1.8 x 10^-7
  (0.2,10^-5,1%)                    -> B(Z -> bs) < 4.2 x 10^-7
  (0.2,10^-5,0.1%)                  -> B(Z -> bs) < 4.2 x 10^-8

Supplementary SM-width extract:
Using the total Z width Gamma_Z = 2.4952 +/- 0.0023 GeV, the paper translates
partial widths to:

  B(Z -> bs) = (4.2 +/- 0.7) x 10^-8
  B(Z -> bd) = (1.8 +/- 0.3) x 10^-9.
