Title: Top flavour-changing neutral interactions: theoretical expectations
and experimental detection
Author: J. A. Aguilar-Saavedra
Submitted: 2004-09-29; last revised: 2005-07-06, version v4
Journal reference: Acta Phys. Polon. B35:2695-2710, 2004
arXiv DOI: 10.48550/arXiv.hep-ph/0409342

Relevant abstract excerpt:

Top flavour-changing neutral interactions with a light quark q = u,c and a
gauge or Higgs boson are very suppressed within the Standard Model, but can
reach observable levels in many of its extensions. The paper reviews possible
sizes of the effective vertices Ztq, gamma tq, gtq and Htq.

Relevant source-table excerpt:

In the source file rev.tex, Table 1 gives branching ratios for top FCN decays
in the SM and in several SM extensions. The table entries include:
  t -> u Z: SM 8 x 10^-17
  t -> c Z: SM 1 x 10^-14
  t -> c Z: quark-singlet model 1.1 x 10^-4
  t -> c Z: 2HDM approximately 10^-7
  t -> c Z: FC 2HDM approximately 10^-10
  t -> c Z: MSSM 2 x 10^-6
  t -> c Z: R-parity-violating SUSY 3 x 10^-5

The same source text states before the table:
  Br(t -> cZ) approximately 1 x 10^-14,
  Br(t -> cH) approximately 3 x 10^-15.
