Title:
Search for tau- -> mu- mu+ mu- decays at the LHCb experiment with Run 2 data.

Authors:
R. Aaij et al. (LHCb Collaboration).

Metadata:
Submitted to arXiv on 28 Jan 2026.
Report numbers: LHCb-PAPER-2025-052, CERN-EP-2026-003.

Relevant context:
The search uses LHCb Run 2 pp data at sqrt(s) = 13 TeV and targets the
all-muon companion mode tau- -> mu- mu+ mu-. It does not provide a
tau- -> e- e+ e- bound. L010 therefore remains B-factory dominated in the
current catalog value, while the companion L009 has an active LHCb program.

Use in L010:
Context-only source to address the Belle/BaBar/LHCb landscape requested for
the tau -> three-lepton LFV entries. No L010 numerical value is taken from
this source.
