Title:
Search for flavour-changing neutral-current interactions of a top quark and a
gluon in pp collisions at sqrt(s)=13 TeV with the ATLAS detector

Authors: ATLAS Collaboration
Submitted: 2021-12-02; last revised: 2022-07-01, version v2
Journal reference: Eur. Phys. J. C 82 (2022) 334
DOI: 10.1140/epjc/s10052-022-10182-7
arXiv DOI: 10.48550/arXiv.2112.01302

Relevant abstract values:
- The analysis searches for single-top production via left-handed top-gluon
  FCNC interactions u+g -> t and c+g -> t.
- Dataset: pp collisions at sqrt(s)=13 TeV with integrated luminosity
  139 fb^{-1}.
- Selection: exactly one electron or muon, exactly one b-tagged jet, and
  missing transverse momentum.
- Observed data are consistent with the background-only hypothesis.
- 95% CL limits:
  sigma(u+g -> t) * B(t -> Wb) * B(W -> l nu) < 3.0 pb.
  sigma(c+g -> t) * B(t -> Wb) * B(W -> l nu) < 4.7 pb.
- The W leptonic branching fraction used for the sum over e, mu, tau modes is
  B(W -> l nu) = 0.325.
- EFT interpretation:
  |C_uG^{ut}| / Lambda^2 < 0.057 TeV^{-2}.
  |C_uG^{ct}| / Lambda^2 < 0.14 TeV^{-2}.
- Equivalent FCNC top-decay branching fractions:
  B(t -> u + g) < 0.61e-4.
  B(t -> c + g) < 3.7e-4.
