Title: Search for the decay modes D0 -> e+e-, D0 -> mu+mu-, and D0 -> e mu
Authors: The BABAR Collaboration

Updated: 2012-08-01
Journal reference: Phys. Rev. D 86, 032001 (2012)

Comments: 12 pages, 4 figures

arXiv API abstract snapshot:
  BABAR searched for D0 -> e+e-, D0 -> mu+mu-, and D0 -> e mu in continuum
  e+e- -> c cbar events using a data sample corresponding to 468 fb^-1.
  The observed event yields were consistent with expected backgrounds, except
  for a D0 -> mu+mu- upward background fluctuation at the 5% level. Using the
  Feldman-Cousins method, BABAR set 90% confidence-level intervals:
  B(D0 -> e+e-) < 1.7 x 10^-7, B(D0 -> mu+mu-) in [0.6, 8.1] x 10^-7, and
  B(D0 -> e mu) < 3.3 x 10^-7.

Extraction note:
  This post-2008 search is weaker than the Belle 2010 D0 -> e+ e- limit but
  is retained as an independent experimental cross-check and PDG-listed input.
