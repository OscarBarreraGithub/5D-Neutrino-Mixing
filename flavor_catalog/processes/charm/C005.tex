\section{C005: \(D^0\to e^+e^-\)}

\subsection*{Process}
\textbf{Standard notation:}
\(\mathcal{B}(D^0\to e^+e^-)\).
This entry covers the rare neutral-charm leptonic decay
\(D^0\to e^+e^-\), a \(\Delta C=1\), \(c\to u e^+e^-\)
flavor-changing neutral-current probe.  Charge-conjugate modes are implicit in
the experimental listings.  It is the electron-helicity counterpart to C004,
\(D^0\to\mu^+\mu^-\), but has a separate experimental history.

\subsection*{PDG / equivalent value(s)}
The current PDG Live/API entry S032.39, accessed on 2026-05-16, gives
\[
  \mathcal{B}(D^0\to e^+e^-) < 7.9\times10^{-8}
  \qquad (90\%~{\rm C.L.}).
\]
The PDG-used input is the Belle 2010 search with \(660\,{\rm fb}^{-1}\) at
KEKB.  Belle reports the same \(7.9\times10^{-8}\) upper limit and no signal
evidence.  The independent BaBar 2012 search used \(468\,{\rm fb}^{-1}\) and
set the weaker \(1.7\times10^{-7}\) upper limit at the same confidence level.
These numbers are traced to
\texttt{flavor\_catalog/references/C005/pdg2026\_d0\_ee\_pdgLive.txt},
\texttt{belle2010\_arxiv1003\_2345.txt}, and
\texttt{babar2012\_arxiv1206\_5419.txt}.  The dispatch note called this an
LHCb upper limit, but the accessed PDG/API measurement list contains no LHCb
\(D^0\to e^+e^-\) entry.

\subsection*{Relevance to the RS / anarchic-flavor pipeline}
Anarchic warped models can induce up-sector flavor-changing neutral currents
through nonuniversal KK gauge couplings, flavor-violating \(Z\)-like effects,
Higgs/radion scalar exchange, or lepton-sector extensions.  C005 therefore
tests a pure \(c\to u\) dilepton amplitude rather than the four-quark
\(\Delta F=2\) operator basis used for neutral \(D\)-mixing.  For the current
quark-only scan it is a catalog and roadmap process; a live veto would need
both the \(c\to u\) current and electron-current normalization.

\subsection*{Post-2008 developments}
The arXiv:0804.1954 RS-flavor baseline predates the Belle and BaBar leptonic
\(D^0\) searches.  The main post-2008 experimental development is Belle's
2010 improvement to the present PDG limit, followed by BaBar's 2012
cross-check in a different \(e^+e^-\to c\bar c\) sample.  Unlike C004,
which has newer hadron-collider dimuon searches, the PDG-accessed C005 record
still uses the Belle result as the headline input.  Rare-charm theory work
continues to emphasize that purely leptonic \(D^0\) limits are clean null
tests experimentally but require care when separating short-distance new
physics from long-distance charm dynamics.

\subsection*{Constraint validity and model dependence}
The upper limit itself is experimentally robust.  Its interpretation as an RS
constraint is model dependent because the electron mode is helicity sensitive,
bremsstrahlung and electron identification enter the experimental acceptance,
and the Standard Model rare-charm amplitude is not a precision short-distance
prediction.  A conservative use is an upper bound on additional short-distance
\(c\to u e^+e^-\) amplitudes in a specified operator basis, not a direct
subtraction against a sharply predicted Standard Model rate.

\subsection*{Implementation difficulty}
\textbf{HIGH.}  Production use would require a new \(\Delta C=1\) rare
leptonic module with vector, axial-vector, scalar, and pseudoscalar
normalizations, electron-mode acceptance conventions, and a documented
long-distance treatment.  The existing \(\Delta F=2\) SLL/SLR/VLL/VRR/LR
basis is not enough.
