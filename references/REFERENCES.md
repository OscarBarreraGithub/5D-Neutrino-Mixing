# Reference plots per constraint (RS flavor / EWPT reproduction)

Compiled 2026-06-22. Goal: for each of the 8 user constraints, the most
authoritative published paper that contains a **reproducible plot** of that
bound in a warped/RS (or composite-Higgs) model, plus the experimental inputs
that paper used so we can show the **shift** when we redo it with current data.

PDFs are in `references/papers/`. BibTeX keys are in `references/refs.bib`.
Implementation-ready axis/curve specs are in `references/PLOT_SPECS.md`.

Conventions used across these papers:
- First gauge KK mass `M_g(1) = x1 * Lambda_IR`, with `x1 ~ 2.45` -> `M_g(1) ~ 2.45 * M_KK`. Papers quote bounds sometimes in `M_KK`, sometimes in `M_g(1)`; always check which.
- `v = 174 GeV` (our convention); some papers use `246 GeV`. Watch factors of sqrt(2).

## Data-availability flags (read before reproduction)

Every constraint exposes a per-draw **`ratio` = NP/bound** (ratio<1 passes) and
**`params.M_KK`** [GeV]. These suffice for the simple "`ratio` vs M_KK scatter +
`ratio=1` line" version of every plot. The richer literature figures need extra
quantities NOT stored as numerics:

- **EW001 (S-T ellipse, Fig. 4 of 0807.4937):** needs actual **(S, T)** per draw. Only `ratio_EW001` + a text description of the proxy coefficients are stored. **Recompute** S = c_S v^2/M_KK^2 (c_S=30, U=0) and Delta T per model (minimal: pi*L/(2 c_W^2); custodial: -pi/(4 c_W^2 L)) from M_KK; see the `needs_human_physics` string in each EW001 row.
- **T010/T011 ((g_L^b,g_R^b) plane, Fig. 8 of 0807.4937):** needs **delta g_L^b, delta g_R^b** per draw. Only the R_b/A_b ratios are stored. **Recompute** from the Zbb adapter (gauge-KK + fermion-admixture m_b^2/M_KK^2 pieces), or fall back to ratio-vs-M_KK.
- **C001/C002 (D-mixing region, Fig. 1 of 0906.1879):** needs **x12^NP/x** and CP phase **sin(2 sigma_D)** per draw. C002 stores `proxy_flags` (q_over_p_deviation_proxy, phi_d_sine_deviation_proxy) which are close but 1-D; the 2-D region needs the full M12^NP magnitude + phase. **Extract/recompute.**
- **Collider (Fig. 16 CMS):** the figure is a **mass cut**, not a per-draw ratio. Use `params.M_KK` directly against the 5.5 TeV (CMS) / 4.1 TeV (ATLAS) exclusion. The CRxxx constraints already encode the experimental mass edges.
- **mu->e gamma (L001):** `ratio_L001` = BR/limit is rigorous; BR = ratio * 1.5e-13 (current MEG II default). Plot BR vs M_KK and draw both the 1.5e-13 and the paper-era 1.2e-11 lines.
